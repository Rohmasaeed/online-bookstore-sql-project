# Online Bookstore SQL Project

## Project Overview
This project demonstrates **SQL database design and analytics** using an **Online Bookstore dataset**. The database consists of three main tables: **Books, Customers, and Orders**.  

The goal is to **analyze sales, revenue, customer behavior, and inventory** using SQL queries.

---

## Features / Business Insights
- Top-selling books and genres 📚  
- Total revenue generated 💵  
- Customers with multiple orders 👥  
- Top 5 customers by spending 💰  
- Remaining stock after fulfilling orders 📦  
- Books that were never ordered ❌  

---

## SQL Skills Demonstrated
- SELECT, WHERE, ORDER BY  
- JOINs (INNER, LEFT)  
- GROUP BY & HAVING  
- Aggregate functions: SUM, AVG, COUNT  
- COALESCE, advanced filtering  
- Ranking & LIMIT  

---

## Sample Queries
- Total revenue from all orders  
- Most frequently ordered books  
- Top 3 most expensive Fantasy books  
- Sales by genre  
- Top 5 customers by total spending
